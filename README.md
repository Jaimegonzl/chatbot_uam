## Instrucciones de Instalación

1. Clonar o descargar el repositorio.
2. Crear el entorno virtual: `python -m venv venv` y activarlo.
3. Instalar dependencias: `pip install -r requirements.txt`
4. Configurar variables de entorno: Copia el archivo `.env.example`, renombralo como `.env` y genera tu propia clave secreta.
5. Ejecutar la aplicación: `python app.py`
